## Description
The `number_pattern` function takes a single argument `n` and returns
a string containing numbers from `1` up to `n`.

The function also handles invalid inputs by returning clear
error messages.

## Function Behaviour

### Valid input
```python
number_pattern(4)
Output:

1 2 3 4

Invalid input (not an integer)
number_pattern(0.5)


Output:

Argument must be an integer value.

Invalid input (non-positive integer)
number_pattern(0)
